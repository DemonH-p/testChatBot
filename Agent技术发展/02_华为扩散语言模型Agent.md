# 华为发布业界首个扩散语言模型Agent：部分场景提速8倍

## 核心观点

华为诺亚方舟实验室发布业界首个**扩散语言模型Agent（DLLM Agent）**，在完全相同的Agent工作流、训练数据和交互预算下，执行速度提升30%以上，部分复杂任务中跑出**8倍于传统自回归(AR)模型**的效率。

## 技术背景

当Agent的框架、工具、数据和训练方式都保持一致时，仅仅改变语言模型的生成范式（Autoregressive vs Diffusion），会系统性地改变Agent的规划与行为模式。

## 核心发现

### 性能优势

| 指标 | DLLM Agent表现 |
|------|---------------|
| 执行速度 | 提升30%以上 |
| 复杂任务 | 部分场景提速8倍 |
| 规划收敛 | 更早收敛到正确轨迹 |
| 冗余操作 | 回溯和冗余更少 |
| 交互轮次 | 使用更少交互轮次和工具调用 |

### 深层原因

这种优势**并非仅来自并行解码速度**，而是体现在Agent级别的规划与决策行为上：

- DLLM展现更强的Planner能力
- 更早收敛到正确轨迹
- 回溯和冗余操作更少

## 技术意义

1. **范式创新**：首次验证扩散模型在Agent场景的系统性优势
2. **重新定义评估标准**：Agent能力不仅看"答对问题"，更要看用最短路径、最少交互预算完成任务
3. **工程化价值**：为生产级Agent提供新的模型选择路径

## 展望

扩散语言模型可能成为Agent架构的新基建，特别在需要快速响应、多轮交互的场景中具有明显优势。
